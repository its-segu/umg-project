# UMG MGMT-Project

Sample Project for Artst: MGMT

### Demo

```
npm install
npm start
```

App will start on localhost:3000.

# Apis/Libraries

- Spotify API
- Framer motion
- React-page-scroller
- Axios
- Material UI


# Troubleshooting

```
npm install react react-dom --save
```
